# README_IMPORT

## Бърз старт
1. Разархивирай файловете
2. Качи всичко в ново GitHub repo: `SharkPilot-store`
3. В Cloudflare Pages:
   - New Project
   - Connect to Git
   - избери repo-то
   - Framework preset = None
   - Build output directory = public

## Първи файлове, които може да редактираш
- `public/data/products.json`
- `public/data/discount-codes.json`
- `public/assets/img/products/`
- `public/assets/img/hero/hero-bg.png`

## Важни бележки
- BG е език по подразбиране
- количката се пази в `localStorage`
- ако правиш тестове и виждаш стари продукти, изчисти browser storage
